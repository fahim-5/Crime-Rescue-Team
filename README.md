# 🚔 Crime Rescue Project – Live Crime Reporting Platform

## 🎯 Project Overview
The **Crime Rescue Project** is an innovative **real-time crime reporting platform** that enhances public safety through **community validation and automated police notifications**. It minimizes false reports and ensures a rapid police response using a **crowdsourced verification system**.

---

## 🏗 Project Workflow & Features

### 🛡️ **User Registration & Authentication**
✔️ **Secure NID-based registration** to ensure authenticity.  
✔️ **Token-based authentication** for safe access.  
✔️ **Prevents fake accounts** with identity validation.

### 🚨 **Crime Reporting System**
✔️ **Users can report crimes** in real time via a structured form.  
✔️ **Crime details include**:
   - **Type:** Theft, robbery, assault, harassment, etc.
   - **Location:** GPS auto-detection or manual entry.
   - **Description:** Detailed incident report.
   - **Evidence:** Images or videos (optional).
✔️ **Reports are broadcasted to nearby users** instantly.

### 📍 **Geo-Based Notifications**
✔️ Users **within a 1 km radius** receive instant alerts.  
✔️ Encourages **local community validation**.  
✔️ **Real-time engagement** for verification.

### 🏛️ **Community Validation System**
✔️ Users can **visit the crime scene** and validate incidents.  
✔️ **Validation requires submission of proof** (e.g., images, descriptions, or videos).  
✔️ **After 5+ confirmations**, the report is **marked as verified**.

### 👮 **Automated Police Alert Mechanism**
✔️ **Once validated by 5+ users**, an **alert is sent to the police**.  
✔️ Police officers receive:
   - **Crime Location & Details**
   - **Validation Evidence from Users**
   - **Live Feedback Updates**
✔️ Police can **accept, track, and update case status**.

### 🏢 **Admin Panel for Police Verification**
✔️ **Police officers apply separately for verification**.  
✔️ **Admins review and approve police accounts**.  
✔️ **Admins monitor all crime reports and system logs**.

## 🛠 **Tech Stack & Implementation**

### 🎨 **Frontend (User Interface)**
- ⚡ **React.js** – Fast, responsive UI.
- 🔄 **Redux.js** – Efficient global state management.
- 📌 **Google Maps API** – Location-based services.
- 🛰️ **Socket.io** – Real-time notifications.

### 🏗 **Backend (Server & APIs)**
- 🚀 **Node.js & Express.js** – Scalable API development.
- 🔐 **JWT Authentication** – Secure user sessions.
- 🛰️ **Socket.io** – Real-time communication.

### 🗄 **Database & Storage**
- 💾 **MySQL** – Stores user data, reports, and police logs.
- ☁️ **AWS S3 / Firebase Storage** – Media evidence storage.
